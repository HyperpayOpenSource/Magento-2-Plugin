== Description ==
This is the official Gate2Play.com payment gateway plugin for WooCommerce.

= How to use it: =
Having installed and activated the plugin, go to the WooCommerce settings page and select the Payment Gateways tab. Here you'll be able to select and configure the payment gateway.

Once that's done, it will tie in with WooCommerce automatically.

= Compatability =
The plugin has been tested with WooCommerce 2.1.2 I can't guarantee that it will work with older versions of WooCommerce.
